# 제주 설화 소스 자료

## 폴더 구조

```
sources/
├── README.md              # 이 파일
├── web-crawled/           # Claude가 웹에서 수집한 자료
│   └── *.md
├── pdf-downloads/         # 사용자가 다운로드한 PDF
│   └── *.pdf
└── source-list.md         # 수집 대상 목록 및 상태
```

## 수집 원칙

1. 출처 명시 필수
2. 원문 최대한 보존
3. 구조화된 포맷(마크다운/JSON)으로 저장
