# 제주 설화 데이터 스키마

## 개요
제주도를 5km × 5km 격자로 나눠 각 지역의 설화/스토리를 매핑하기 위한 데이터 구조

---

## 테이블 구조

### 1. grids (격자 테이블)
| 필드 | 타입 | 설명 | 예시 |
|------|------|------|------|
| `grid_id` | string | 격자 고유 ID | `SS-B2` |
| `region` | string | 권역명 | `성산권` |
| `lat_min` | float | 위도 최소 | `33.44` |
| `lat_max` | float | 위도 최대 | `33.49` |
| `lng_min` | float | 경도 최소 | `126.90` |
| `lng_max` | float | 경도 최대 | `126.95` |
| `area_name` | string | 지역 통칭 | `성산리·오조리 일대` |

### 2. spots (스팟 테이블)
| 필드 | 타입 | 설명 | 예시 |
|------|------|------|------|
| `spot_id` | string | 스팟 고유 ID | `SPOT-001` |
| `grid_id` | string | 소속 격자 (FK) | `SS-B2` |
| `name` | string | 스팟명 | `성산일출봉` |
| `type` | enum | 유형 | `관광지/마을/오름/해안/문화재` |
| `lat` | float | 위도 | `33.458` |
| `lng` | float | 경도 | `126.942` |
| `description` | string | 간단 설명 | `UNESCO 세계자연유산` |

### 3. stories (설화 테이블)
| 필드 | 타입 | 설명 | 예시 |
|------|------|------|------|
| `story_id` | string | 설화 고유 ID | `STR-001` |
| `spot_ids` | array | 연관 스팟들 (FK) | `["SPOT-001", "SPOT-005"]` |
| `title` | string | 설화 제목 | `등경돌 전설` |
| `category` | enum | 유형 | `창조신화/지명유래/역사전설/의례/인물` |
| `summary` | string | 요약 (1-2문장) | `설문대할망이 바느질할 때...` |
| `content` | text | 상세 내용 | (전체 스토리) |
| `characters` | array | 등장인물 | `["설문대할망", "김통정"]` |
| `source` | string | 출처 | `제주도 전설지 (현용준)` |
| `keywords` | array | 키워드 | `["설문대할망", "바느질", "등잔"]` |

---

## 관계도

```
grids (1) ──────< (N) spots
                      │
                      └──< (M:N) stories
```

---

## Category (설화 유형) Enum 값

| 값 | 설명 | 예시 |
|---|---|---|
| `창조신화` | 제주도/자연물 생성 이야기 | 설문대할망 한라산 창조 |
| `지명유래` | 지명이 생긴 이유 | 우도(소섬), 쇠머리오름 |
| `역사전설` | 역사적 사건/인물 기반 | 김통정 삼별초, 하멜 표류 |
| `의례` | 제사, 굿, 신앙 관련 | 영등굿, 잠수굿 |
| `인물` | 전설적 인물 이야기 | 고득종, 오현 |
| `자연전설` | 동식물, 자연현상 설화 | 용 전설, 도깨비 이야기 |

---

## Spot Type Enum 값

| 값 | 설명 |
|---|---|
| `관광지` | 공식 관광 명소 |
| `마을` | 마을, 리 단위 |
| `오름` | 제주 오름 |
| `해안` | 해변, 포구, 해안절벽 |
| `문화재` | 지정 문화재, 유적지 |
| `자연` | 숲, 동굴, 폭포 등 |
