# 본풀이 (제주 무속 신화)

> 출처: 디지털제주문화대전 (jeju.grandculture.net)
> 항목 ID: GC00700916
> 크롤링 일시: 2024-12-03

## 정의

"본풀이"는 제주의 심방(무당)들이 굿을 진행할 때 제상 앞에 앉아 노래하는 신의 내력담이다.
- "본" = 근본
- "풀이" = 해석, 설명

신의 출생부터 신으로서의 직능을 갖추고 좌정하기까지의 과정을 담고 있다.

## 세 가지 분류 체계

### 1. 일반신 본풀이

사람의 일반적 사항을 담당하는 신들의 이야기.

**주요 종류:**
| 본풀이 | 담당 영역 |
|--------|----------|
| 천지왕 본풀이 | 천지 창조, 일월 조정 |
| 삼승할망 본풀이 | 출산, 육아 |
| 마누라 본풀이 | 무병장수 |
| 초공 본풀이 | 무조신(巫祖神) |
| 이공 본풀이 | 꽃(영혼의 환생) |
| 삼공 본풀이 | 전상(前生) |
| 차사 본풀이 | 저승 |
| 세경 본풀이 | 농경 |
| 지장 본풀이 | 중음(中陰) |
| 문전 본풀이 | 가택(집) |
| 칠성 본풀이 | 뱀신 |
| 군웅 본풀이 | 수렵 |

### 2. 당신 본풀이

각 마을의 당신(지역 수호신)에 관한 이야기.

**주요 당신:**
- 송당
- 궤눼기당
- 토산
- 일뤳당
- 여드렛당

**특징:**
- 구성 길이와 문예적 완성도가 다양
- 신명, 직능, 단골, 식성 등을 기록

### 3. 조상신 본풀이

각 집안의 행운과 안녕을 보호하는 신에 관한 이야기.

**특징:**
- 혈연조상뿐 아니라 뱀, 원혼 등도 포함
- 아들→아들 또는 딸→딸로 전승
- 석살림 때 노래하며 집안의 행운을 기원

## 문화적 의의

- 조상신 본풀이를 통해 집안의 번영을 기원
- 무속신앙의 핵심적 신화 자료
- 제주 고유의 구비문학 형식
