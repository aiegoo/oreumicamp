# Jeju Stories Source Archive Branch

This branch (`jeju-sources-archive`) preserves the original PDF source files from the Jeju folklore collection project.

## Branch Purpose
- **Archive**: Preserve original 480+ PDF source files
- **Source Management**: Maintain crawled data and conversion tools
- **Historical Reference**: Keep track of data collection methodology

## Contents

### PDF Sources
- **Location**: `team-data/jeju-stories/sources/pdf-downloads/`
- **Total Files**: 480 PDF documents
- **Categories**: 
  - `신화/` (Myths): Original mythological stories
  - `민담/` (Folk tales): Traditional folk stories
  - `failed_downloads.txt`: Log of failed download attempts

### Tools & Scripts
- **Location**: `team-data/jeju-stories/sources/`
- **Crawlers**: JavaScript files for web scraping
- **Converters**: PDF to JSON conversion utilities
- **Documentation**: Data classification and conversion guides

## Original Data Collection
Based on commit history, this collection included:
- **Total PDF Sources**: 1,555 files (before reorganization)
- **Myths**: 182 PDFs (C_M, T_M, W_M prefixes)
- **Legends**: 1,051 PDFs (C_L, T_L, W_L prefixes) 
- **Folk tales**: 322 PDFs (C_F, T_F, W_F prefixes)
